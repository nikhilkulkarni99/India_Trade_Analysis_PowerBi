# India Import-Export Trade Analysis (2021–2023)

## 📊 Project Overview - 
- This is a real world raw dataset taken from government of India's site (data.gov.in). This Power BI dashboard analyzes India's import-export trade data across principal commodities and countries for 2021–2023.

## 🎯 Objectives - 
- Analyze trade trends over time
- Identify top commodities driving trade
- Understand country-wise trade distribution
- Evaluate trade balance and dependency patterns

## 📁 Dataset - 
- Principal Commodity-wise Import & Export Data (2021–2023)
- Source: data.gov.in

## 📊 Dashboard Features - 
- KPI summary (Export, Import, Trade Balance, etc.)
- Trend analysis (Export vs Import)
- Commodity-level insights
- Country-level analysis
- Interactive slicers

## 🔍 Key Insights - 
- Imports dominate total trade, indicating a trade deficit
- Petroleum-related commodities contribute significantly
- Trade is concentrated among a few key countries
- Export growth observed year-over-year

## 🛠 Tools Used - 
- Power BI
- DAX
- Data Cleaning & Transformation
